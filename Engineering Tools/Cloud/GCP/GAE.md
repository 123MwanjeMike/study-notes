
# Google App Engine

Platform as a Service.
App Engine manages the hardware and networking infrastructure required to run code.
- Easy deployment, maintenance and scalability
- Provides
	- No-SQL databases
	- In-Memory caching
	- Load Balancing
	- Health Checks
	- Logging
	- Authentication
